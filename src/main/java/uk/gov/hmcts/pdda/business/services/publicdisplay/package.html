This package contains the Session Beans and associated classes for the public displays. The two session beans have separate responsibilities.

PublicDisplayConfigurationBean is responsible for managing the public display configuration data. It presents methods both for the maintenance of the data by a client and the querying of the data by the render component. The transformation from the highly normalised entities addressed by the administrative user managing the configuration and the de-normalised entities used by the render component is handled hear by the DisplayRotationSetDataHelper.
